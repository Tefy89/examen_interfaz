# Comandos GITHUB

# Inicilizar el repositorio local
```
git init
```
- Referencia del repositorio local con el repositorio remoto (GITHUB)
```
git remote add origin https://github.com/Tefy89/dise-o_interfaz.git
```
# Validar Referencia
```
remote -v 
```

```
git add .
git commit -m "proyecto base"
